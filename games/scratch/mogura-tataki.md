# モグラ叩きゲームを作ろう！

## 今回作るゲーム

今回は「モグラ叩きゲーム」を作っていきます。魔法の杖を動かして、ランダムに出てくるキャラクターたちを叩きましょう。

### 操作方法
- **矢印キー**: 魔法の杖の移動
- **スペースキー**: 叩く

---

## ゲームを作る準備

### 1. 初期設定
1. スプライト1は使わないので削除します
2. 背景を設定します

### 2. 必要なスプライト
以下のスプライトを用意しましょう：
- 魔法の杖（プレイヤーが操作するキャラクター）
- モグラやキャラクター（叩かれる対象）

---

## プログラミング手順

### ステップ1: 魔法の杖のプログラム

魔法の杖を矢印キーで動かせるようにします。

#### 基本的な動き
```
🏁 クリックされたとき
ずっと
  もし <右向き矢印キー が押された> なら
    x座標を 10 ずつ変える
  end
  もし <左向き矢印キー が押された> なら
    x座標を -10 ずつ変える
  end
  もし <上向き矢印キー が押された> なら
    y座標を 10 ずつ変える
  end
  もし <下向き矢印キー が押された> なら
    y座標を -10 ずつ変える
  end
end
```

#### 叩く動作
```
🏁 クリックされたとき
ずっと
  もし <スペース キーが押された> なら
    大きさを 120 % にする
    0.1 秒待つ
    大きさを 100 % にする
  end
end
```

---

### ステップ2: モグラのプログラム

モグラがランダムに出現し、叩かれたら消えるようにします。

#### ランダム出現
```
🏁 クリックされたとき
ずっと
  隠す
  1 から 3 までの乱数 秒待つ
  x座標を -200 から 200 までの乱数 、y座標を -150 から 150 までの乱数 にする
  表示する
  1 秒待つ
end
```

#### 叩かれた時の処理
```
🏁 クリックされたとき
ずっと
  もし <魔法の杖 に触れた> かつ <スペース キーが押された> なら
    隠す
    [得点 v] を 1 ずつ変える
  end
end
```

---

### ステップ3: 得点システム

#### 変数を作る
1. 「変数」から「変数を作る」をクリック
2. 「得点」という名前の変数を作成

#### 得点の初期化
```
🏁 クリックされたとき
[得点 v] を 0 にする
```

---

## 発展的な機能

### タイマーを追加
制限時間を設定してゲームを盛り上げましょう。

```
🏁 クリックされたとき
[時間 v] を 30 にする
30 回繰り返す
  1 秒待つ
  [時間 v] を -1 ずつ変える
end
すべてを止める
```

### 難易度を上げる
- モグラの出現速度を速くする
- モグラの表示時間を短くする
- モグラの数を増やす（スプライトを複製）

### サウンドを追加
- 叩いた時に効果音を鳴らす
- BGMを追加する

```
もし <魔法の杖 に触れた> かつ <スペース キーが押された> なら
  [ポップ v] の音を鳴らす
  隠す
  [得点 v] を 1 ずつ変える
end
```

---

## 完成イメージ

- 魔法の杖を矢印キーで自由に動かせる
- スペースキーで叩く動作ができる
- ランダムに出現するモグラを叩くと得点が入る
- 制限時間内にできるだけ高得点を目指す

---

## チャレンジ課題

1. **複数のモグラを出現させよう**
   - スプライトを複製して、同時に複数のモグラが出るようにする

2. **ボーナスキャラクターを追加しよう**
   - 特定のキャラクターを叩くと得点が2倍になる

3. **ミスした時のペナルティを追加しよう**
   - 叩いてはいけないキャラクターを追加し、間違えると減点

4. **難易度選択を追加しよう**
   - ゲーム開始時に「かんたん」「ふつう」「むずかしい」を選べるようにする

---

## まとめ

このゲームで学んだこと：
- キー入力によるキャラクター操作
- 乱数を使ったランダムな動き
- 変数を使った得点管理
- 条件分岐（もし〜なら）の使い方
- タイミング制御（〜秒待つ）

これらの技術を応用すれば、様々なゲームを作ることができます！
