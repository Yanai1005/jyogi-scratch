# HTMLの基礎を学ぼう！

## HTMLとは？

**HTML**（エイチティーエムエル）は、**HyperText Markup Language**の略で、ウェブページの「骨組み」を作る言語です。

Scratchでいうと、**スプライトや背景を配置する**のと似ています。HTMLでは、文字や画像、ボタンなどをウェブページに配置します。

---

## HTMLの基本構造

HTMLは「タグ」というものを使って書きます。タグは `<` と `>` で囲まれた記号です。

### 基本的なHTMLファイル

```html
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="UTF-8">
  <title>私のウェブページ</title>
</head>
<body>
  <h1>こんにちは！</h1>
  <p>これは私の最初のウェブページです。</p>
</body>
</html>
```

### 各部分の説明

- `<!DOCTYPE html>`: これはHTMLファイルだよ、と宣言する
- `<html>`: HTMLの始まりと終わり
- `<head>`: ページの情報（タイトルなど）を書く場所
- `<title>`: ブラウザのタブに表示されるタイトル
- `<body>`: 実際にページに表示される内容を書く場所

---

## よく使うHTMLタグ

### 見出し（タイトル）

```html
<h1>一番大きな見出し</h1>
<h2>2番目に大きな見出し</h2>
<h3>3番目に大きな見出し</h3>
```

数字が大きくなるほど、文字が小さくなります。

### 段落（文章）

```html
<p>これは段落です。文章を書くときに使います。</p>
```

### ボタン

```html
<button>クリックしてね！</button>
```

### 画像

```html
<img src="cat.png" alt="かわいい猫の写真">
```

- `src`: 画像ファイルの場所
- `alt`: 画像が表示されないときの説明文

### リンク

```html
<a href="https://scratch.mit.edu/">Scratchのサイトへ</a>
```

### リスト

```html
<ul>
  <li>りんご</li>
  <li>バナナ</li>
  <li>みかん</li>
</ul>
```

## HTMLを書いてみよう！

### ステップ1: ファイルを作る

1. メモ帳やテキストエディタを開く
2. 新しいファイルを作る
3. `index.html` という名前で保存

### ステップ2: コードを書く

```html
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="UTF-8">
  <title>自己紹介ページ</title>
</head>
<body>
  <h1>自己紹介</h1>
  <p>名前: 太郎</p>
  <p>好きなもの: ゲーム、プログラミング</p>
  
  <h2>好きな食べ物</h2>
  <ul>
    <li>カレー</li>
    <li>ラーメン</li>
    <li>アイスクリーム</li>
  </ul>
  
  <button>クリック！</button>
</body>
</html>
```

### ステップ3: ブラウザで開く

保存したファイルをダブルクリックすると、ブラウザで表示されます！

---

## Scratchとの比較

| Scratch | HTML |
|---------|------|
| スプライトを配置 | タグで要素を配置 |
| 「〇〇と言う」ブロック | `<p>` タグで文字を表示 |
| 背景を設定 | `<body>` の中に内容を書く |
| コスチューム | `<img>` タグで画像を表示 |

## 練習問題

自分の好きなものを紹介するページを作ってみよう！

1. 好きなゲームやアニメの名前を `<h1>` で書く
2. 説明を `<p>` で書く
3. 好きなキャラクターのリストを `<ul>` と `<li>` で作る

## まとめ

- HTMLはウェブページの「骨組み」を作る
- タグを使って、文字や画像、ボタンを配置する
- `<h1>`, `<p>`, `<button>` などのタグがある
- ファイルを作ってブラウザで開けば、すぐに確認できる
