# JavaScript ゲーム講座

JavaScriptを使ったゲーム開発を学びましょう！

Scratchで学んだプログラミングの考え方を、本物のプログラミング言語で表現してみよう！

---

## 📚 学習の流れ

### ステップ1: 基礎を学ぼう

まずは、ウェブページを作るための3つの言語を学びます。

#### 1. [HTMLの基礎](./basics-html.md)
- ウェブページの「骨組み」を作る
- 文字やボタンを配置する方法
- Scratchの「スプライトを配置する」のと似ています

#### 2. [CSSの基礎](./basics-css.md)
- ページを「かっこよく」する
- 色や大きさを変える方法
- Scratchの「色の効果を変える」のと似ています

#### 3. [JavaScriptの基礎](./basics-javascript.md)
- ページに「動き」を付ける
- ボタンをクリックしたときに何かを起こす
- Scratchの「ブロックを組み合わせる」のと同じです

---

### ステップ2: ゲームを作ろう

基礎を学んだら、実際にゲームを作ってみましょう！

#### [モグラ叩きゲームを作ろう！](./mogura-tataki.md)
- 30秒間でモグラを叩くゲーム
- HTML、CSS、JavaScriptを全部使います
- Scratchで作ったゲームと同じものを作ります

---

## 🎮 JavaScriptでゲームを作るといいこと

### 1. **本物のプログラミング**
Scratchで学んだことが、プロも使う本物のプログラミング言語で書けるようになります！

### 2. **自由にカスタマイズ**
色、形、動き、すべて自分の好きなように変えられます。

### 3. **みんなに見せられる**
完成したゲームは、インターネットで友達や家族に見せることができます！

### 4. **将来に役立つ**
JavaScriptは、世界中で使われている人気のプログラミング言語です。

## 💡 学習のコツ

1. **順番に進めよう**: HTMLの基礎 → CSSの基礎 → JavaScriptの基礎 → ゲーム作り
2. **実際に書いてみよう**: 読むだけじゃなくて、自分でコードを書いてみることが大事！
3. **分からなくても大丈夫**: 最初は難しく感じるかもしれないけど、少しずつ慣れていきます
4. **楽しもう**: ゲームを作るのは楽しい！自分だけのゲームを作ってみよう！
