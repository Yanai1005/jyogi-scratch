# JavaScript ゲーム講座

JavaScriptを使ったゲーム開発を学びましょう！Scratchの基本を理解した方が、実際のプログラミング言語で同じゲームを作ることで、より深い理解が得られます。

## 講座一覧

### まずは基礎を学ぼう

ゲームを作る前に、ウェブ開発の基礎を学びましょう！

- [HTMLの基礎](./basics-html.md)
  - ウェブページの「骨組み」を作る
  - タグの使い方
  - Scratchとの比較で理解しやすい

- [CSSの基礎](./basics-css.md)
  - ページを「かっこよく」する
  - 色や大きさの変え方
  - ボタンのスタイリング

- [JavaScriptの基礎](./basics-javascript.md)
  - ページに「動き」を付ける
  - 変数と関数の使い方
  - イベント処理の基本

### ゲームを作ろう

基礎を学んだら、実際にゲームを作ってみましょう！

- [モグラ叩きゲームを作ろう！](./mogura-tataki.md)
  - HTML/CSS/JavaScriptでの実装
  - DOM操作の基本
  - イベントハンドリング
  - ゲーム状態の管理

## JavaScriptゲーム開発の利点

- **実際のウェブ技術**: 学んだことがそのまま実用的なウェブアプリに
- **カスタマイズ性**: デザイン・機能・ロジック全てを自由に実装
- **本格的なプログラミング**: 構造化されたコードを書く力が身につく
- **デプロイ可能**: 完成したゲームをすぐにオンラインで公開可能

## 前提知識

- HTMLの基本的な理解
- CSSで簡単なスタイリングができること
- JavaScriptの変数と関数の基本概念

これらがまだの場合は、各講座で丁寧に説明します。

それでは、JavaScriptでのゲーム開発を始めましょう！
