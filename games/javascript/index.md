# JavaScript ゲーム講座

JavaScriptを使ったゲーム開発を学びましょう！Scratchの基本を理解した方が、実際のプログラミング言語で同じゲームを作ることで、より深い理解が得られます。

## 講座一覧

### 初心者向け

- [モグラ叩きゲームを作ろう！](./mogura-tataki.md)
  - HTML/CSS/JavaScriptでの実装
  - DOM操作の基本
  - イベントハンドリング
  - ゲーム状態の管理

## JavaScriptゲーム開発の利点

- **実際のウェブ技術**: 学んだことがそのまま実用的なウェブアプリに
- **カスタマイズ性**: デザイン・機能・ロジック全てを自由に実装
- **本格的なプログラミング**: 構造化されたコードを書く力が身につく
- **デプロイ可能**: 完成したゲームをすぐにオンラインで公開可能

## 前提知識

- HTMLの基本的な理解
- CSSで簡単なスタイリングができること
- JavaScriptの変数と関数の基本概念

これらがまだの場合は、各講座で丁寧に説明します。

それでは、JavaScriptでのゲーム開発を始めましょう！
